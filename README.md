# Monsun
A travelling agent system implemented in C++14.

# Requirements
- CMake (>= 3.14)
- Make
- Any compiler that support C++14
- Google Test (optional)

# Supported Platforms
- Windows
- Linux
- MacOS (untested)

# Installation
## Windows
Follow the CMake GUI instructions

## Linux and MacOS
```sh
make && sudo make install
```

# Uninstall
## Windows
Follow the CMake GUI instructions

## Linux and MacOS
```sh
sudo make uninstall
```

# To Run
## Windows
Find the application named `monsun` from your menu

## Linux and MacOS
```sh
monsun
```
